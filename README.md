# SignalTrust Smart-City Data Governance Project

## Project Overview

This repository supports our SignalTrust Smart-City Backbone project for MSDS 485: Data Governance, Ethics, and Law.

SignalTrust is a smart-city data governance case. In this project, a telecom company provides 5G infrastructure for a new smart district. The smart district uses IoT sensors, public Wi-Fi, citizen service data, and broadband infrastructure data.

The main goal of this project is to show how smart-city data can be managed in a safe, ethical, and legal way.

## Project Goals

This project focuses on:

- Exploratory Data Analysis (EDA)
- Data quality review
- PII identification
- Data masking
- Data classification
- Role-based access control (RBAC)
- Bias and equity analysis
- Data lineage
- Data governance documentation
